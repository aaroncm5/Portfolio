Projects

Pokemon Ultimate Team

Pokemon Ultimate Team is a full stack application that utilizes ReactJS to create the front end experience. I used NodeJS to create the server that allows the front end side of the site to communicate with the Database I created using MySQL.

The premise of the site is to allow the user to create various combinations of teams and compare the various statistics and move combinations allowed. The site uses user authentication with basic email and password verification as well as online session storage to allow users to save and team created to a database created in order to view teams created at a later time.
